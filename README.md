# Telegram Mini App "Лотерея"

Monorepo проект для Telegram Mini App "Лотерея".

## Структура
- **frontend/** — Vite + TypeScript фронтенд
- **backend/** — FastAPI бэкенд
- **docker-compose.yml** — для контейнеризации

## Запуск
- Инициализируйте git-репозиторий: `git init`
- Смотрите отдельные README в папках frontend и backend (после их создания)
